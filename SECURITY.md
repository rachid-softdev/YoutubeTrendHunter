# Security Policy

## Reporting a Vulnerability

We take security seriously. If you discover a security vulnerability, please email us at **security@trendhunter.app**.

**Do not** open public GitHub issues for security vulnerabilities.

## Process

1. Report the vulnerability via email.
2. We will acknowledge receipt within 48 hours.
3. We will work on a fix and keep you informed of progress.
4. Once the fix is deployed, we will credit you in the release notes (if desired).

## Scope

- Web application at https://trendhunter.app
- Chrome extension "TrendHunter"
- Public APIs at *.trendhunter.app

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest  | ✅ Yes |
| Older   | ❌ No |
